concept:

an E-commerce website

Description:

Cloud is an online store which sells stationeries,accessories and stickers Moreover,for each order , a tree will be planted in the region.

Features used in this project:

-Carousel:from boostrap
-Sign in modal:HTML/CSS and Js used from W3schools
-Add to cart button:using HTML/CSS and Js (by stocking data from orders page to the localstorage and retrieving it from the cart)
-A cart:using HTML and CSS
-Total price:using Js DOM
-Deleting Items from cart using Js
-A counter to count trees each time checkout button is clicked using HTML/CSS and Js
